# Zed 在Windows上的构建

仅构建官方源码，您可以fork本仓库自行构建，检查action workflow文件以确认安全性。

每天构建官方最新源码，可以在release或者action的artifact中找到构建完毕的zed程序。

本仓库接受对官方源码的patch，以实现官方未及时处理或合并的代码。

patch文件将在构建过程中自动应用
